Данные Московского Марш-Броска и программы для их обработки

install  -- скрипт для копирования куда следует всего, что нужно
            выкладывать на www
html/    -- положения, объявления о месте старта и т.п. (static html)
logo/    -- эмблемы
scripts/ -- cgi-скрипты регистрации
regdata/ -- работа с данными регистрации. Исходных данных здесь нет,
            так как они содержат адреса участников
results/ -- подсчет результатов
rating/  -- подсчет рейтинга
proto/   -- бланк протокола
